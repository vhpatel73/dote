Real-time Integration Guide

Technical Specs How to report AI benefits via Webhook API?

Overview

The AI Value Board provides a REST-compatible webhook endpoint that allows external AI applications to report their realized benefits automatically. This enables real-time tracking of productivity gains and revenue impact without manual entry.

Endpoint Specification

POST {{ webhook_url }}

Content-Type: application/json

Payload Structure

Field Type Requirement Description
webhook_key String Required The secure secret key for the specific initiative.
kpi_value Float Conditional Required if the initiative tracks Productivity Gain.
revenue_impact Float Conditional Required if the initiative tracks New Business.
month String Optional Format: YYYY-MM. Defaults to current month if omitted.

Interactive Example

Use the following code snippet to integrate within your Python application:

python_integration.py 
import requests
import json

url = "{{ webhook_url }}"
payload = {
    "webhook_key": "{% if initiative %}{{ initiative.webhook_key }}{% else %}YOUR_SECRET_KEY{% endif %}",
    "kpi_value": 150.5,           # Replace with your actual KPI metric
    "revenue_impact": 0,         # Optional based on initiative type
    "month": "2024-03"           # Optional (defaults to current)
}

response = requests.post(url, json=payload)
print(response.json())
